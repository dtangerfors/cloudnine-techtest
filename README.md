## Getting Started
### Installation

_To install this repo, please follow the guide below._

1. Clone the repo
   ```sh
   git clone https://github.com/dtangerfors/cloudnine-techtest.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
